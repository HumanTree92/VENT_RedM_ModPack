# About VENT_RedM_ModPack
This is a Mod Pack for Velociti Entertainments Public RedM Server.

# Download & Installation:
1) Download the .zip.
2) Extract the .zip or Open the .zip.
3) Go to your RedM Application Data Folder (RedM Directory).
4) Drag & Drop all the contents inside either the Simple Drag & Drop or Advanced Drag & Drop into your RedM Directory.
